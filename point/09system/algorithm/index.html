# 排序

  1. 平方阶 (O(n2)) 排序 各类简单排序：直接插入、直接选择和冒泡排序。
  2. 线性对数阶 (O(nlog2n)) 排序 快速排序、堆排序和归并排序;
  3. O(n1+§)) 排序, § 是介于 0 和 1 之间的常数。 希尔排序
  4. 线性阶 (O(n)) 排序 基数排序, 此外还有桶、箱排序。

关于稳定性 : 排序后 2 个相等键值的顺序和排序之前它们的顺序相同

  1. 稳定的排序算法：冒泡排序、插入排序、归并排序和基数排序。
  2. 不是稳定的排序算法：选择排序、快速排序、希尔排序、堆排序。

# 荷兰旗问题

输入乱序排列的三色小球, 通过两两交换, 达到分组效果

  • 3个指针head, middle (指向头) ; tail (指向尾); 0表示红, 1表示白, 2表示蓝
  • middle指针所指元素为0时, 与head指针所指的元素交换, 而后middle++, head++
  • middle指针所指元素为1时, 不做任何交换（即球不动）, 而后middle++
  • middle指针所指元素为2时, 与tail指针所指的元素交换, 而后, middle指针不动, tail--

给定x, 将无序数组重构为以下结构: ① 小于等于x || 大于x ② 小于x || 等于x || 大于x

  • 设定区域, 并根据情况扩张区域 / 移动下标
  • ①选择设定一个区域: 小于等于区域
  • ②选择设定两个区域: 小于区域 和 大于区域

# 堆排序

  • 创建一个堆 H[0……n-1]

  • 先把最大的数浮到堆顶 (升序, 小顶堆), 将目前这个堆称为h1

    • 此处浮到堆顶, 只需要检查所有非叶子节点
    1. 把h1堆首和堆尾互换, 此时堆尾肯定是最大值, 队尾元素可以固定, 不再考虑
    2. 将堆的尺寸缩小 1 作为新的堆 h1, 调整顶端数据, 让堆顶的元素再次作为最大值
      • 此处调整, 只需要将堆顶的向下沉到大小合适的位置即可
    3. 重复1和2; 直到 h1 只剩1个元素

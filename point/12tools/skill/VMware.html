# 安装VMware Tools

方式1

VMwareTools已经在vmware安装目录（C:\Program Files (x86)\VMware\VMware Workstation）下装载镜像便可以获取，将其复制出来，安装

方式2

  • 点击菜单 虚拟机 ——> 安装VMware Tools(T)...
  • 不存在 /mnt/cdrom 则创建 mkdir -p /mnt/cdrom
  • 挂载目录mount -t auto /dev/cdrom /mnt/cdrom
  • 拷贝安装包到用户家目录 cp /mnt/cdrom/VMwareTools-XXX .
  • 解除挂载 umount /dev/cdrom
  • 解压安装包 tar -zxvf VMwareTools-XXX
  • 安装VMware Tools
    • cd vmware-tools-distrib/
    • ./vmware-install.pl

# CentOS 安装 mysql

  1. mysql 8+

    • 参考 centos7安装MySQL8.0.16 (opens new window)

    1. 清理环境, 删除原有的mysql和mariadb

    2. 安装mysql官网提供的mysql repo源 (opens new window)

      • wget http：//dev.mysql.com/get/Downloads/mysql80-community-release-el7-1.noarch.rpm
      • rpm -ivh mysql80-community-release-el7-1.noarch.rpm
      • yum install mysql-community-server

    3. 启动mysqld服务，并设为开机自动启动

      • systemctl start mysqld.service
      • systemctl enable mysqld.service

    4. 修改 /etc/my.cnf 并重启

      • default-authentication-plugin=mysql_native_password
      • symbolic-links=0
      • skip-grant-tables #跳过密码
      • validate-password=OFF #直接关闭密码强度(只能用于mysql5.7, mysql8没有)

    5. 修改密码, 远程登录

      1. update user set authentication_string=’’ where user=‘root’; --将默认密码置空后去配置文件取消跳过密码
      2. alter user ‘root’@‘localhost’ identified by ‘123’; --这里有密码强度校验
      3. 远程登录
        1. update user set host='%' where user='root'; --设置root用户所有地址可登录
        2. alter user 'root'@'%' identified by '123456' password expire never;--将用户root密码设置为永不过期
      4. flush privileges;

    6. 顺便打开centos 防火墙端口

      • 开启端口, 查询端口号是否开启, 哪些端口开启, 不要忘了重启生效
      1. firewall-cmd --zone=public --add-port=3306/tcp --permanent
      2. firewall-cmd --query-port=3306/tcp
      3. firewall-cmd --list-port

  2. mysql 5.7

    1. 安装centos 只选择网络项, 安装时候可以在图形化安装页面把网络配置好
    2. 和mysql 8 同理, 安装官网的yum repo
      • wget https://dev.mysql.com/get/mysql57-community-release-el7-9.noarch.rpm
      • rpm -ivh mysql57-community-release-el7-9.noarch.rpm
      • yum install mysql-community-server
    3. 设置密码和远程等后续工作同上

======== windows cmd 使用http代理 =========

环境：shadowsocks windows 本地ss端口设置(这里1080)

cmd命令行:(不用socks5)(临时设置)(也可放置环境变量) set http_proxy=http://127.0.0.1:1080 set https_proxy=http://127.0.0.1:1080

ps:一定要用cmd命令行，千万别用powershell !!! 简易测试命令：curl https://www.google.com（别用ping）

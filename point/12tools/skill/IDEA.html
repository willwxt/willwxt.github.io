# IDEA教程

基于尚硅谷2018版教程

  • IDEA是JETBRAINS旗下产品, 主流的Java开发环境, 使用很多, 还介绍了jetbrains其他开发工具, Android Studio是基于IDEA社区版开发迭代的
  • 企业开发,web开发,Java,Scala,out of the box 开箱即用
  • 支持很多语言(部分通过插件实现),SQL,框架,容器
  • 相比Eclipse,更广泛强大的语言提示功能支持(SQL,web),无Eclipse的js/jsp文件编译bug

# 版本

Ultimate Community
JavaEE,框架,JavaWeb开发,DATABASE 主要支持桌面, 安卓

  • 点击下面 Compare (opens new window) 可以查看对比的详细差异

  • 点击上面IDEA菜单栏的 What's New (opens new window) 查看历史版本

Eclipse使用ABCD开头的单词命名版本, IDEA使用年份+数字命名(2020.1, April 9)

  • 最好安装在SSD上, 安装后自动安装1.8jre, 文件目录

    • 安装目录
      • bin 启动文件, 虚拟机配置, 属性配置信息
        • 对应虚拟机配置信息,
        • Xms设置JVM启动时初始内存, 提高启动速度
        • Xmx设置运行分配最大内存, 降低垃圾回收频率
        • ReservedCodeCacheSize保留代码缓存大小
      • help 帮助文件
      • lib 依赖类库
      • license 证书
      • plugins 插件
    • 用户>$(user)>IntelliJ IDEA
      • config 配置文件, 里面有keymaps plugins 等 类似于eclipse的.metadata
      • system 里面的caches是缓存
    • 工程目录

  • IDEA 执行时会生产大量的缓存, 索引文件 (体验更好, 内存消耗更大)

  • 工程下的src类似于Eclipse下的src目录, 用于存放代码

  • 工程下的.idea 和 project01.iml 文件都是IDEA特有的, 类似于Eclipse工程下的.settings .classpath .project 等

# 模块

官网 :

  • IDEA的project相当于eclipse的workspace
  • IDEA的module相当于eclipse的project
  • 但是IDEA一个窗口不能有多个project, 一般大项目都是多模块的

新建module

删除module的方式 : 右键无直接删除选项, 打开Project Structure 工程结构, 进行删除, 此时会将对应模块修改成普通文件夹, 然后再对其右键删除

Project Structure 工程结构

  • Project Settings
    • Project
      • Project compiler output : 编译后class文件输出的目录

# 常用配置

  • Appearance & Behavior

    • Appearance 主题

  • Editor

    • Color Scheme 颜色方案 编辑区主题

      • Console Font 单独设置控制台字体, 同样编辑区也可以单独设置
      • Language Defaults
        • Comments 设置注释的样式
          • Line comment 单行注释
          • Doc comment 文档注释

    • Code Style

      • Java
        • Imports

    • General 滚轮改变字体大小 Change font size with Ctrl+Mouse Wheel

      • Auto Import 自动导包

      • Appearance 显示行号, 显示方法分隔符

      • Code Editing 鼠标移入文本提示 Show quick documentation on mouse move

      • Code Completion 提示不区分大小写(Match Case)

    • File and Code Templates 可以设置文件头信息

  • Build, Execution, Deployment

    • Compiler 编译
      • Build project automatically 自动编译

  • Keymap 快捷键

    • 和Eclipse还有不同 可以通过导入jar包实现大部分快捷键同步 File > Manage IDE Settings > Import Settings
    • 搜索栏, 可以按对应功能搜索快捷键
    • 搜索栏边上的按钮, 可以按快捷键搜索对应的功能

File > Power Save Mode 省电模式, 会关闭代码检查和代码提示

右键文件栏, Split Vertically, Split Horizontally 水平垂直分栏

# 断点调试

  1. Debug 的优化设置
    • Settings > Build, Execution, Deployment > Debugger > Java > Transport: win系统勾选 Shared memory
  2. 几个操作, Step Over , Step Into , Resume
  3. 条件断点, 在断点上点击右键设置条件, 循环时候会执行到断点满足的时候 (i=60)
  4. 查看已有变量(表达式)的值, Ctrl+u

# 其他设置

# 设置 javaDoc
  • 对Module 或 具体文件
  • Tools > Generate JavaDoc > 选择整个工程, 还是模块, 还是文件 > 选择生成路径(Output Dir)
  • Locale 语言(zh_CN)
  • Other command line arguments 设置一些编码集 -encoding UTF-8 -charset UTF-8
  • 点击ok, 生成文档
# 缓存和索引的清理
  • IDEA首次加载会生产索引, 加快查询
  • IDEA的缓存和索引文件也会损坏, 需要清除缓存和索引文件
    • File > Invalidate Caches > 第一项 Invalidate and Restart
    • 或者删除system文件夹
    • 本地的历史记录 local history 会被清除, 位置在 用户>nampy>.idea>system>localhistory
# 关闭IDEA自动更新
  • Settings > Appearance & Behavior > System Settings > Updates > Automatically check updates

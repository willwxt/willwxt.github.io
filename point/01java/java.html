# String变量何时放入常量池

String放入运行时常量池的时机与String.intern()方法解惑 (opens new window)

String b = "计算机" + "软件"; //编译期间会优化为上面a这句

String c = new String("计算机软件");

# intern() 方法

作用:

  • 如果常量池中已经包含此String对象内容的字符串, 则返回常量池中该字符串的引用
  • 如果没有, 则在常量池中创建与此String内容相同的字符串, 并返回

概括: str.intern() 查询(创建)并返回常量池对象

jdk1.7 改变: 如果没有在常量池中查询到, 则在常量池中记录Java堆中首次出现的该字符串的引用
